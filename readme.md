# Googleカレンダー✖️ZoomAPI連携で個別相談の予約を自動化

## DEMO

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - GoogleカレンダーのスケジュールをWEBブラウザに反映

  - 空き時間から個別相談の日時を選択して予約→Googleカレンダーにも反映

  - 予約と同時にZoomリンクが発行、WEBブラウザに表示される
